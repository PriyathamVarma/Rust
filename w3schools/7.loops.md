# 🦀 Rust Loops

## 1. 📌 What are Loops?

Loops are used to **repeat a block of code multiple times**.

### 💡 Why use loops?
- Saves time ⏱️  
- Reduces code repetition 🔁  
- Makes code cleaner and readable 👀  

---

## 2. 🔄 Types of Loops in Rust

Rust has **3 types of loops**:

1. `loop` → infinite loop  
2. `while` → runs while condition is true  
3. `for` → runs fixed number of times  

---

## 3. 🔁 `loop` (Infinite Loop)

Runs forever unless stopped manually.

### Example:

```rust
fn main() {
    loop {
        println!("This will repeat forever!");
    }
}
````

⚠️ Use `Ctrl + C` to stop the program.

---

### 🛑 Breaking a Loop

Use `break` to stop the loop.

```rust
fn main() {
    let mut count = 1;

    loop {
        println!("Hello World!");

        if count == 3 {
            break;
        }

        count += 1;
    }
}
```

### 🔍 Explanation:

* Starts from `count = 1`
* Runs until `count == 3`
* Prints 3 times

---

### 🔢 Returning Value from Loop

You can return a value using `break`.

```rust
fn main() {
    let mut count = 1;

    let result = loop {
        println!("Hello!");

        if count == 3 {
            break count;
        }

        count += 1;
    };

    println!("The loop stopped at: {}", result);
}
```

💡 `break value` → returns value from loop

---

## 4. 🔄 `while` Loop

Runs **as long as condition is true**.

### Example:

```rust
fn main() {
    let mut count = 1;

    while count <= 5 {
        println!("Count: {}", count);
        count += 1;
    }
}
```

### Output:

```
1 2 3 4 5
```

---

### ❌ When Condition is False Initially

```rust
fn main() {
    let count = 10;

    while count <= 5 {
        println!("This won't run.");
    }
}
```

💡 Loop will **not execute even once**

---

### 🛑 Break in while Loop

```rust
fn main() {
    let mut num = 1;

    while num <= 10 {
        if num == 6 {
            break;
        }

        println!("Number: {}", num);
        num += 1;
    }
}
```

💡 Prints 1 to 5 only

---

### ⏭️ Continue in while Loop

```rust
fn main() {
    let mut num = 1;

    while num <= 10 {
        if num == 6 {
            num += 1;
            continue;
        }

        println!("Number: {}", num);
        num += 1;
    }
}
```

💡 Skips number 6

---

## 5. 🔁 `for` Loop

Used when you know **how many times to loop**.

### Example:

```rust
fn main() {
    for i in 1..6 {
        println!("i is: {}", i);
    }
}
```

### 🔍 Explanation:

* `1..6` → from 1 to 5
* 6 is **not included**

---

### ✅ Inclusive Range

Use `..=` to include last value.

```rust
fn main() {
    for i in 1..=6 {
        println!("i is: {}", i);
    }
}
```

💡 Prints 1 to 6

---

### ⚙️ Break and Continue in for Loop

```rust
fn main() {
    for i in 1..=10 {
        if i == 3 {
            continue; // skip 3
        }

        if i == 5 {
            break; // stop loop
        }

        println!("i is: {}", i);
    }
}
```

### Output:

```
1 2 4
```

---

## 6. 📊 Loop Comparison

| Loop Type | Use Case                   | Behavior                  |
| --------- | -------------------------- | ------------------------- |
| loop      | Unknown repetitions        | Runs forever until break  |
| while     | Condition-based repetition | Runs while condition true |
| for       | Fixed iterations / ranges  | Runs specific times       |

---

## 7. 🔑 Important Keywords

* `break` → stops loop 🛑
* `continue` → skips current iteration ⏭️

---

## 🧠 Quick Revision

* `loop` → infinite loop 🔁
* `while` → runs on condition 🔄
* `for` → fixed iterations 🔢
* `break` → stop loop
* `continue` → skip iteration
