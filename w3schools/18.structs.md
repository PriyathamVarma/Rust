# 🦀 Rust Structs

## 1. 📌 What is a Struct?

A struct (short for **structure**) is a **custom data type** used to group related values together.

💡 Think of it like:
- A **real-world object** 🧍  
- Example: Person → name, age, can_vote  

---

## 2. 🛠️ Creating a Struct

Use the `struct` keyword and define fields inside `{}`.

### Example:

```rust
struct Person {
    name: String,
    age: u32,
    can_vote: bool,
}
````

### 🔍 Explanation:

* `name`, `age`, `can_vote` → fields
* Fields are like variables inside a struct

---

## 3. 📦 Creating a Struct Instance (Object)

You can create an object from the struct.

### Example:

```rust id="q7m2bx"
struct Person {
    name: String,
    age: u32,
    can_vote: bool,
}

fn main() {
    let user = Person {
        name: String::from("John"),
        age: 35,
        can_vote: true,
    };

    println!("Name: {}", user.name);
    println!("Age: {}", user.age);
    println!("Can vote? {}", user.can_vote);
}
```

---

## 4. 🔍 Access Struct Fields

Use dot `.` syntax.

```rust
user.name
user.age
user.can_vote
```

---

## 5. ✏️ Modify Struct Fields

Make the struct **mutable (`mut`)** to change values.

### Example:

```rust id="8k1v9r"
struct Person {
    name: String,
    age: u32,
}

fn main() {
    let mut user = Person {
        name: String::from("John"),
        age: 35,
    };

    user.age = 36;

    println!("Name: {}", user.name);
    println!("Updated age: {}", user.age);
}
```

---

## 6. 🧠 Fields vs Variables

* Variables → standalone
* Fields → belong to a struct

💡 Fields are part of a **bigger structure**

---

## 7. 🎯 Why Use Structs?

* 🧩 Group related data together
* 👀 Improve readability
* 🔧 Make code easier to maintain
* 🌍 Represent real-world entities (user, car, book)

---

## 🧠 Quick Revision

* `struct` → custom data type
* Fields → variables inside struct
* Use `{}` to define values 📦
* Access using `.` 🔍
* Use `mut` to modify ✏️
