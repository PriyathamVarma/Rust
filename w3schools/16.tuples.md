# 🦀 Rust Tuples

## 1. 📌 What are Tuples?

A tuple is a **collection of multiple values of different types** stored in a single variable.

💡 Useful when you want to:
- Group different types together 🧩  
- Return multiple values from a function 📤  

---

## 2. 🛠️ Create a Tuple

Use parentheses `()` and separate values with commas.

### Example:

```rust
fn main() {
    let person = ("John", 30, true);
}
````

### 🔍 Explanation:

* `"John"` → string (`&str`)
* `30` → integer (`i32`)
* `true` → boolean (`bool`)

---

## 3. 🔢 Access Tuple Values

Use dot `.` followed by index.

### Example:

```rust id="r8n3k2"
fn main() {
    let person = ("John", 30, true);

    println!("Name: {}", person.0);
    println!("Age: {}", person.1);
    println!("Is active: {}", person.2);
}
```

---

## 4. 📦 Packing a Tuple

Storing values in a tuple is called **packing**.

```rust id="6wz9p1"
let person = ("Jenny", 45, false);
```

---

## 5. 📤 Unpacking a Tuple

Extract values into separate variables.

### Example:

```rust id="k3p8xq"
fn main() {
    let person = ("Jenny", 45, false);

    let (name, age, active) = person;

    println!("Name: {}", name);
    println!("Age: {}", age);
    println!("Active: {}", active);
}
```

💡 This is called **destructuring**

---

## 6. 🔁 Return Tuple from Function

Tuples are commonly used to return **multiple values**.

### Example:

```rust id="z4b2mn"
fn get_user() -> (String, i32) {
    (String::from("Liam"), 25)
}

fn main() {
    let user = get_user();

    println!("User: {} ({} years old)", user.0, user.1);
}
```

---

## 🧠 Quick Revision

* Tuple = multiple values (different types) 🧩
* Use `()` to create
* Access with `.index` 🔢
* Unpack using `(a, b, c)` 📤
* Useful for returning multiple values 🔁
