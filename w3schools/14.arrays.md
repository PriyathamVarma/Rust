# 🦀 Rust Arrays

## 1. 📌 What are Arrays?

Arrays are used to **store multiple values in a single variable**.

💡 Instead of creating many variables, you can group them together.

---

## 2. 🛠️ Create an Array

Use square brackets `[]` and separate values with commas.

### Example:

```rust
fn main() {
    let numbers = [1, 2, 3, 4, 5];
}
````

### ⚠️ Important Rule:

* All elements must be of the **same type**

---

## 3. 🔢 Access Array Elements

Use **index** to access values.

* Index starts from `0`

---

### Example:

```rust id="a9z6pw"
fn main() {
    let numbers = [1, 2, 3, 4, 5];

    println!("The first number is: {}", numbers[0]);
}
```

---

## 4. ✏️ Change Array Values

Make array **mutable (`mut`)** to modify values.

### Example:

```rust id="v6h5lt"
fn main() {
    let mut numbers = [1, 2, 3, 4, 5];

    numbers[0] = 10;

    println!("The new first number is: {}", numbers[0]);
}
```

---

## 5. 📏 Array Length

Use `.len()` to get number of elements.

### Example:

```rust id="l7zj3q"
fn main() {
    let numbers = [1, 2, 3, 4, 5];

    println!("This array has {} elements.", numbers.len());
}
```

---

## 6. 🔁 Loop Through an Array

Use `for` loop to iterate over elements.

### Example:

```rust id="q8b7tw"
fn main() {
    let fruits = ["apple", "banana", "orange"];

    for fruit in fruits {
        println!("I like {}.", fruit);
    }
}
```

---

## 7. 🖨️ Print Arrays

### 🔹 Print Single Element

```rust id="h9k2pl"
println!("{}", numbers[0]);
```

---

### 🔹 Print Entire Array

```rust id="2yq9xn"
println!("{:?}", numbers);
```

### ⚠️ Important:

* `{}` → for single values
* `{:?}` → for full data structures (arrays, vectors)

---

## 8. ⚖️ Fixed Size vs Dynamic Size

### 🔒 Arrays (Fixed Size)

* Cannot grow or shrink ❌
* Size is fixed at creation

```rust
fn main() {
    let mut cars = ["Volvo", "BMW", "Ford"];

    // ❌ Error: index out of bounds
    // cars[3] = "Mazda";
}
```

---

### 📈 Vectors (Dynamic Size)

* Can grow and shrink ✅

```rust id="2g7h4m"
fn main() {
    let mut cars = vec!["Volvo", "BMW", "Ford"];

    cars.push("Mazda");

    println!("{:?}", cars);
}
```

---

## 🧠 Quick Revision

* Arrays store multiple values 📚
* Index starts from `0` 🔢
* Use `mut` to modify ✏️
* `.len()` → get size 📏
* `{}` → single value
* `{:?}` → entire array 🖨️
* Arrays = fixed size 🔒
* Vectors = dynamic size 📈
