# 🦀 Rust Data Types

## 1. 📘 What are Data Types?

Data types define **what kind of value a variable can store**.

💡 In Rust:
- You **do NOT need to manually specify types** (usually)
- Rust automatically detects the type → called **Type Inference**

---

## 2. 🤖 Type Inference (Automatic Type Detection)

Rust decides the type based on the value assigned.

### Example:

```rust
fn main() {
    let my_num = 5;         // integer
    let my_double = 5.99;   // float
    let my_letter = 'D';    // character
    let my_bool = true;     // boolean
    let my_text = "Hello";  // string
}
````

### 🔍 Explanation:

* `5` → integer
* `5.99` → floating-point number
* `'D'` → single character
* `true` → boolean
* `"Hello"` → string

---

## 3. 🧾 Explicit Type Declaration

You can also **manually specify types**.

### Example:

```rust
fn main() {
    let my_num: i32 = 5;
    let my_double: f64 = 5.99;
    let my_letter: char = 'D';
    let my_bool: bool = true;
    let my_text: &str = "Hello";
}
```

### 💡 Why use this?

* Improves clarity
* Required in some advanced cases

---

## 4. 📊 Basic Data Types in Rust

| Category  | Type | Description                   | Example      |
| --------- | ---- | ----------------------------- | ------------ |
| Numbers   | i32  | Whole numbers                 | 10, -5       |
| Numbers   | f64  | Decimal numbers               | 5.99         |
| Character | char | Single character              | 'A'          |
| String    | &str | Text (sequence of characters) | "Hello"      |
| Boolean   | bool | True or false                 | true / false |

---

## 5. 🔢 Number Types

### 5.1 Integer (`i32`)

* Stores **whole numbers**
* Can be positive or negative

```rust
fn main() {
    let age: i32 = 25;
    println!("Age is: {}", age);
}
```

---

### 5.2 Floating Point (`f64`)

* Stores **decimal numbers**

```rust
fn main() {
    let price: f64 = 19.99;
    println!("Price is: ${}", price);
}
```

---

## 6. 🔤 Character (`char`)

* Stores a **single character**
* Must use **single quotes `' '`**

```rust
fn main() {
    let my_grade: char = 'B';
    println!("{}", my_grade);
}
```

---

## 7. 📝 Strings (`&str`)

* Stores **text (multiple characters)**
* Must use **double quotes `" "`**

```rust
fn main() {
    let name: &str = "John";
    println!("Hello, {}!", name);
}
```

---

## 8. 🔘 Boolean (`bool`)

* Stores only:

  * `true` ✅
  * `false` ❌

```rust
fn main() {
    let is_logged_in: bool = true;
    println!("User logged in? {}", is_logged_in);
}
```

---

## 9. 🔗 Combining Data Types

You can use multiple data types in the same program.

### Example:

```rust
fn main() {
    let name = "John";
    let age = 28;
    let is_admin = false;

    println!("Name: {}", name);
    println!("Age: {}", age);
    println!("Admin: {}", is_admin);
}
```

---

## 🧠 Quick Revision

* Rust automatically detects types 🤖
* You can also define types manually 🧾
* Main data types:

  * `i32` → integer
  * `f64` → decimal
  * `char` → single character
  * `&str` → string
  * `bool` → true/false
* Strings use `" "` and characters use `' '` ⚠️
