# 🦀 Rust Enums

## 1. 📌 What is an Enum?

An enum (short for **enumeration**) is a type that can be **one of several possible values**.

💡 Each possible value is called a **variant**

---

## 2. 🎯 Why Use Enums?

Enums are useful when:
- A value can only be from a fixed set 🔒  
- You want cleaner and safer code 🛡️  
- You handle multiple cases using `match`  

---

## 3. 🛠️ Creating an Enum

Use the `enum` keyword.

### Example:

```rust
enum Direction {
    Up,
    Down,
    Left,
    Right,
}
````

---

## 4. 📦 Using Enum Values

Use `::` to access variants.

### Example:

```rust id="y3f6zt"
enum Direction {
    Up,
    Down,
    Left,
    Right,
}

fn main() {
    let my_direction = Direction::Up;

    println!("We are going up!");
}
```

---

## 5. 🔄 Match with Enums

Enums work perfectly with `match`.

### Example:

```rust id="p8k2mv"
enum Direction {
    Up,
    Down,
    Left,
    Right,
}

fn main() {
    let my_direction = Direction::Left;

    match my_direction {
        Direction::Up => println!("Going up"),
        Direction::Down => println!("Going down"),
        Direction::Left => println!("Going left"),
        Direction::Right => println!("Going right"),
    }
}
```

💡 Each variant can trigger different behavior.

---

## 6. 📦 Enums with Data

Variants can store extra data.

### Example:

```rust id="d9k3qx"
enum LoginStatus {
    Success(String),
    Error(String),
}

fn main() {
    let result1 = LoginStatus::Success(String::from("Welcome, John!"));
    let result2 = LoginStatus::Error(String::from("Incorrect password"));

    match result1 {
        LoginStatus::Success(message) => println!("Success: {}", message),
        LoginStatus::Error(message) => println!("Error: {}", message),
    }
}
```

### 🔍 Explanation:

* `Success(String)` → stores message
* `Error(String)` → stores error info

---

## 7. 🧠 Enum vs Struct

| Feature | Enum 🧩             | Struct 📦                 |
| ------- | ------------------- | ------------------------- |
| Purpose | One of many values  | Group all values together |
| Example | Direction (Up/Down) | Person (name, age)        |

---

## 🧠 Quick Revision

* `enum` → multiple possible values 🧩
* Variants → options inside enum
* Use `::` to access
* Works well with `match` 🔄
* Can store data inside variants 📦
