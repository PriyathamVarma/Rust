# 🦀 Rust HashMap

## 1. 📌 What is a HashMap?

A HashMap is a **collection of key-value pairs**.

💡 It allows you to:
- Store data using a key 🔑  
- Quickly find values using that key ⚡  

---

## 2. 📦 Import HashMap

Before using it, you must import it:

```rust
use std::collections::HashMap;
````

---

## 3. 🛠️ Create a HashMap

Create an empty HashMap and insert values.

### Example:

```rust
use std::collections::HashMap;

fn main() {
    let mut capital_cities = HashMap::new();

    capital_cities.insert("England", "London");
    capital_cities.insert("Germany", "Berlin");
    capital_cities.insert("Norway", "Oslo");

    println!("{:?}", capital_cities);
}
```

---

## 4. 🔍 Access Values (`get`)

Use `.get()` to access values safely.

### Example:

```rust id="g2x9q4"
use std::collections::HashMap;

fn main() {
    let mut capital_cities = HashMap::new();

    capital_cities.insert("England", "London");
    capital_cities.insert("Germany", "Berlin");
    capital_cities.insert("Norway", "Oslo");

    if let Some(city) = capital_cities.get("England") {
        println!("The capital of England is {}.", city);
    } else {
        println!("England not found.");
    }
}
```

### 🔍 Explanation:

* `.get()` returns `Some(value)` or `None`
* `if let` handles both cases safely

---

## 5. 🔄 Update Values

If key already exists → value is replaced.

### Example:

```rust id="7k8z2p"
use std::collections::HashMap;

fn main() {
    let mut capital_cities = HashMap::new();

    capital_cities.insert("England", "London");
    capital_cities.insert("England", "Berlin");

    println!("{:?}", capital_cities);
}
```

💡 "London" → replaced by "Berlin"

---

## 6. ❌ Remove Values

Use `.remove()` to delete a key.

### Example:

```rust id="5k1b2c"
use std::collections::HashMap;

fn main() {
    let mut capital_cities = HashMap::new();

    capital_cities.insert("England", "London");
    capital_cities.insert("Germany", "Berlin");
    capital_cities.insert("Norway", "Oslo");

    capital_cities.remove("England");

    println!("{:?}", capital_cities);
}
```

---

## 7. 🔁 Loop Through HashMap

Iterate through key-value pairs.

### Example:

```rust id="z9q2v1"
use std::collections::HashMap;

fn main() {
    let mut capital_cities = HashMap::new();

    capital_cities.insert("England", "London");
    capital_cities.insert("Germany", "Berlin");
    capital_cities.insert("Norway", "Oslo");

    for (country, city) in &capital_cities {
        println!("The capital of {} is {}.", country, city);
    }
}
```

---

## 8. ⚠️ Important Rules

* Keys must be **unique** 🔑
* Inserting same key → overwrites value 🔄

---

## 9. 🎯 Why Use HashMaps?

* 🔑 Store data by key
* ⚡ Fast lookup
* 🧩 Organize related data (e.g., name → score)

---

## 🧠 Quick Revision

* HashMap = key-value pairs 🗺️
* Use `insert()` to add
* Use `get()` to access 🔍
* Use `remove()` to delete ❌
* Keys must be unique ⚠️
* Loop using `(key, value)` 🔁
