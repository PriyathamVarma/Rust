# 🦀 Rust Match

## 1. 📌 What is `match`?

`match` is used to **handle multiple conditions (choices)** in a clean and readable way.

💡 It is similar to:
- `switch` in C/Java  
- Multiple `if...else` statements  

👉 Best used when you have **many possible values**

---

## 2. 🛠️ Basic Syntax of `match`

```rust
match value {
    pattern1 => result1,
    pattern2 => result2,
    _ => default_result,
}
````

---

## 3. 📘 Basic Example

```rust
fn main() {
    let day = 4;

    match day {
        1 => println!("Monday"),
        2 => println!("Tuesday"),
        3 => println!("Wednesday"),
        4 => println!("Thursday"),
        5 => println!("Friday"),
        6 => println!("Saturday"),
        7 => println!("Sunday"),
        _ => println!("Invalid day."),
    }
}
```

---

## 4. 🔍 Explanation

* `day` → value being checked
* Each branch:

  * `value => action`
* `_` → default case (runs if no match found)

### Flow:

1. `day` is evaluated once
2. Compared with each branch
3. Matching branch executes

💡 In this example:
`day = 4` → Output: **Thursday**

---

## 5. 🔗 Multiple Matches (`|` operator)

You can match multiple values in a single branch.

### Example:

```rust id="s9f7kc"
fn main() {
    let day = 6;

    match day {
        1 | 2 | 3 | 4 | 5 => println!("Weekday"),
        6 | 7 => println!("Weekend"),
        _ => println!("Invalid day"),
    }
}
```

💡 `|` means **OR condition**

---

## 6. ⚡ match with Return Value

`match` can return a value (like `if...else`).

### Example:

```rust id="3h7qwl"
fn main() {
    let day = 4;

    let result = match day {
        1 => "Monday",
        2 => "Tuesday",
        3 => "Wednesday",
        4 => "Thursday",
        5 => "Friday",
        6 => "Saturday",
        7 => "Sunday",
        _ => "Invalid day.",
    };

    println!("{}", result);
}
```

---

## 7. ⚠️ Type Rule (Important)

All branches must return the **same data type**.

### ❌ Wrong Example:

```rust id="q6b6gr"
let result = match day {
    1 => "Monday",
    2 => 2,
    _ => "Other",
};
```

### ❗ Error:

Different types → string + number ❌

---

### ✅ Correct Example:

```rust id="7kmbt3"
let result = match day {
    1 => "Monday",
    2 => "Tuesday",
    _ => "Other",
};
```

---

## 🧠 Quick Revision

* `match` → cleaner alternative to many `if...else`
* `_` → default case
* `|` → multiple values in one condition
* Can return values ⚡
* All branches must return **same type** ⚠️
