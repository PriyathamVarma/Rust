# 🦀 Rust Vectors (`Vec`)

## 1. 📌 What is a Vector?

A vector is a **resizable array**.

💡 Unlike arrays:
- Arrays → fixed size ❌  
- Vectors → dynamic size ✅  

---

## 2. 🛠️ Creating a Vector

Use the `vec!` macro.

### Example:

```rust
fn main() {
    let fruits = vec!["apple", "banana", "orange"];
}
````

---

## 3. 🔢 Access Vector Elements

Access elements using index (same as arrays).

### Example:

```rust id="u3n2j9"
fn main() {
    let fruits = vec!["apple", "banana", "orange"];

    println!("First fruit: {}", fruits[0]);
}
```

---

## 4. ✏️ Change Vector Values

Make vector **mutable (`mut`)** to modify values.

### Example:

```rust id="f0v2zq"
fn main() {
    let mut fruits = vec!["apple", "banana", "orange"];

    fruits[0] = "grape";

    println!("New first fruit: {}", fruits[0]);
}
```

---

## 5. ➕ Add Elements (`push`)

Adds element to the **end** of the vector.

### Example:

```rust id="4n9n7m"
fn main() {
    let mut fruits = vec!["apple", "banana"];

    fruits.push("cherry");

    println!("{:?}", fruits);
}
```

### Output:

```
["apple", "banana", "cherry"]
```

---

## 6. ➖ Remove Elements (`pop`)

Removes last element.

### Example:

```rust id="6oqz2u"
fn main() {
    let mut fruits = vec!["apple", "banana", "cherry"];

    fruits.pop();

    println!("{:?}", fruits);
}
```

### Output:

```
["apple", "banana"]
```

---

## 7. 📍 Insert at Specific Index

Use `insert(index, value)`.

### Example (Beginning):

```rust id="i8gn3v"
fn main() {
    let mut fruits = vec!["banana", "orange"];

    fruits.insert(0, "apple");

    println!("{:?}", fruits);
}
```

---

### Example (Middle):

```rust id="zt4g2x"
fn main() {
    let mut fruits = vec!["banana", "orange"];

    fruits.insert(1, "apple");

    println!("{:?}", fruits);
}
```

---

## 8. ❌ Remove at Specific Index

Use `remove(index)`.

### Example:

```rust id="1e9mxo"
fn main() {
    let mut fruits = vec!["apple", "banana", "orange"];

    fruits.remove(0);

    println!("{:?}", fruits);
}
```

---

### ⚠️ Performance Note:

* Adding/removing at **beginning or middle** is slower
* Because elements need to shift

---

## 9. 📏 Vector Length

Use `.len()` to get number of elements.

### Example:

```rust id="l6i2dz"
fn main() {
    let fruits = vec!["apple", "banana", "cherry"];

    println!("There are {} fruits.", fruits.len());
}
```

---

## 10. 🔁 Loop Through a Vector

Use `for` loop.

### Example:

```rust id="6ivz9m"
fn main() {
    let fruits = vec!["apple", "banana", "orange"];

    for fruit in &fruits {
        println!("I like {}.", fruit);
    }
}
```

---

### ⚠️ Important (Borrowing in Loop)

* `for fruit in fruits` → moves values ❌
* `for fruit in &fruits` → borrows values ✅

💡 Always use `&` if you want to reuse the vector later.

---

## 🧠 Quick Revision

* `vec![]` → create vector
* Dynamic size 📈
* `push()` → add element
* `pop()` → remove last element
* `insert()` → add at index
* `remove()` → delete at index
* `.len()` → size 📏
* Use `&` while looping 🔁
