# 🦀 Rust Strings

## 1. 📌 What are Strings?

Strings are used to **store text**.

### Example:

```rust
fn main() {
    let greeting: &str = "Hello";
    println!("{}", greeting);
}
````

💡 Strings are always written inside **double quotes `" "`**

---

## 2. 🧩 Types of Strings in Rust

Rust has **two types of strings**:

| Type     | Description                     | Use Case                  |
| -------- | ------------------------------- | ------------------------- |
| `&str`   | String slice (fixed, immutable) | Static text like "Hello"  |
| `String` | Growable, mutable string        | When text needs to change |

💡 In most cases, use `String` because it is more flexible.

---

## 3. 🛠️ Creating a String

You can convert a string literal into a `String`.

### Method 1: `to_string()`

```rust
fn main() {
    let text1 = "Hello World".to_string();
    println!("{}", text1);
}
```

---

### Method 2: `String::from()`

```rust
fn main() {
    let text2 = String::from("Hello World");
    println!("{}", text2);
}
```

💡 Both methods are commonly used.

---

## 4. ✏️ Modifying Strings

To change a string, it must be **mutable (`mut`)**.

---

### 4.1 Add Text (`push_str`)

```rust
fn main() {
    let mut greeting = String::from("Hello");
    greeting.push_str(" World");

    println!("{}", greeting);
}
```

### Output:

```
Hello World
```

---

### 4.2 Add Character (`push`)

```rust
fn main() {
    let mut word = String::from("Hi");
    word.push('!');

    println!("{}", word);
}
```

### Output:

```
Hi!
```

---

## 5. 🔗 Concatenating Strings

### Method 1: `format!` (Recommended ✅)

```rust
fn main() {
    let s1 = String::from("Hello");
    let s2 = String::from("World!");
    let s3 = String::from("What a beautiful day!");

    let result = format!("{} {} {}", s1, s2, s3);

    println!("{}", result);
}
```

💡 Cleaner and easier for multiple strings

---

### Method 2: `+` Operator

```rust
fn main() {
    let s1 = String::from("Hello");
    let s2 = String::from("World!");
    let s3 = String::from("What a beautiful day!");

    let result = s1 + " " + &s2 + " " + &s3;

    println!("{}", result);
}
```

### ⚠️ Important Rule:

* `+` works like: `String + &str`
* That’s why we use `&s2` and `&s3`

---

## 6. 📏 String Length

Use `.len()` to get length.

```rust
fn main() {
    let name = String::from("John");
    println!("Length: {}", name.len());
}
```

### Output:

```
Length: 4
```

---

## 🧠 Quick Revision

* `&str` → fixed string
* `String` → growable string
* Use `mut` to modify strings ✏️
* `push_str()` → add text
* `push()` → add character
* `format!` → best for combining strings 🔗
* `.len()` → get string length 📏
