# 🦀 Rust If..Else Conditions

## 1. 📌 Conditions in Rust

Rust supports standard comparison conditions:

- `<` → Less than  
- `<=` → Less than or equal to  
- `>` → Greater than  
- `>=` → Greater than or equal to  
- `==` → Equal to  
- `!=` → Not equal to  

💡 These conditions are used to **make decisions in programs**.

---

## 2. 🧠 Conditional Statements in Rust

Rust provides the following:

- `if` → execute code if condition is true  
- `else` → execute code if condition is false  
- `else if` → check multiple conditions  
- `match` → handle many cases (advanced, covered later)  

---

## 3. 🔹 if Statement

Executes a block of code **only if condition is true**.

### Example:

```rust
fn main() {
    if 7 > 5 {
        println!("7 is greater than 5.");
    }
}
````

---

### Example with Variables:

```rust
fn main() {
    let x = 7;
    let y = 5;

    if x > y {
        println!("x is greater than y.");
    }
}
```

---

## 4. 🔸 if...else Statement

Executes one block if condition is true, otherwise another block.

### Example:

```rust
fn main() {
    let age = 16;

    if age >= 18 {
        println!("You can vote.");
    } else {
        println!("You are too young to vote.");
    }
}
```

---

## 5. 🔁 else if (Multiple Conditions)

Used to check multiple conditions in sequence.

### Example:

```rust
fn main() {
    let score = 85;

    if score >= 90 {
        println!("Grade: A");
    } else if score >= 80 {
        println!("Grade: B");
    } else if score >= 70 {
        println!("Grade: C");
    } else {
        println!("Grade: F");
    }
}
```

---

## 6. ⚡ Using `if` as an Expression

In Rust, `if...else` can return a value.

### Example:

```rust
fn main() {
    let time = 20;

    let greeting = if time < 18 {
        "Good day."
    } else {
        "Good evening."
    };

    println!("{}", greeting);
}
```

💡 **Important Rule:**
When using `if` as an expression → `else` is mandatory.

---

## 7. ✨ Simplified One-Line Syntax

If blocks contain only one expression:

```rust
fn main() {
    let time = 20;
    let greeting = if time < 18 { "Good day." } else { "Good evening." };

    println!("{}", greeting);
}
```

💡 Works similar to:

* Ternary operator in other languages
* Rust does NOT have `?:`, but this is equivalent

---

## 8. ⚠️ Type Consistency Rule

Both `if` and `else` must return **same data type**.

### ❌ Wrong Example:

```rust
fn main() {
    let number = 5;

    let result = if number < 10 {
        "Too small"
    } else {
        100
    };

    println!("{}", result);
}
```

### ❗ Error:

```
error[E0308]: `if` and `else` have incompatible types
```

### ✅ Correct Example:

```rust
fn main() {
    let number = 5;

    let result = if number < 10 {
        "Too small"
    } else {
        "Big number"
    };

    println!("{}", result);
}
```

---

## 🧠 Quick Revision

* `if` → runs code if condition is true
* `else` → runs if condition is false
* `else if` → multiple conditions
* `if` can return values ⚡
* `else` is required when using `if` as expression
* Both branches must return **same type** ⚠️
