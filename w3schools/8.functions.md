# 🦀 Rust Functions

## 1. 📌 What is a Function?

A function is a **block of code that runs only when it is called**.

### 💡 Why use functions?
- Organize code 🧩  
- Avoid repetition 🔁  
- Improve readability 👀  

---

## 2. 🛠️ Creating a Function

Use the `fn` keyword.

### Syntax:

```rust
fn function_name() {
    // code to be executed
}
````

---

## 3. ▶️ Calling a Function

To run a function, **call it using its name followed by `()`**

### Example:

```rust
fn say_hello() {
    println!("Hello from a function!");
}

fn main() {
    say_hello(); // function call
}
```

---

## 4. 📥 Functions with Parameters

Parameters allow you to **pass data into functions**.

### Example:

```rust
fn greet(name: &str) {
    println!("Hello, {}!", name);
}

fn main() {
    greet("John");
}
```

### 🔍 Explanation:

* `name: &str` → parameter (string input)
* `"John"` → argument passed to function

---

## 5. 📤 Functions with Return Values

Functions can return values using `->`.

### Example (with `return`):

```rust
fn add(a: i32, b: i32) -> i32 {
    return a + b;
}

fn main() {
    let sum = add(3, 4);
    println!("Sum is: {}", sum);
}
```

---

## 6. ⚡ Returning Without `return`

Rust allows **implicit return** (no `return` keyword).

### Example:

```rust
fn add(a: i32, b: i32) -> i32 {
    a + b
}

fn main() {
    let sum = add(3, 4);
    println!("Sum is: {}", sum);
}
```

### ⚠️ Important Rule:

* No semicolon (`;`) on last line
* Last expression is automatically returned

---

## 7. 🔄 Comparison

| Method           | Syntax Style    | Notes            |
| ---------------- | --------------- | ---------------- |
| With `return`    | Explicit return | Clear but longer |
| Without `return` | Implicit return | Short and common |

---

## 8. 🎯 Why Use Functions?

* 🧩 Better code organization
* 🔁 Avoid repeating code
* 👀 Easier to read and maintain

---

## 🧠 Quick Revision

* `fn` → defines function
* Call using `function_name()`
* Parameters → input values 📥
* Return → output values 📤
* Last line without `;` returns value ⚡
